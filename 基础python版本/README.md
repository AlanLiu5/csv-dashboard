# 基础 Python 数据分析

使用原生 Python 实现 CSV 数据分析。

---

## 功能

- CSV 读取
- 清洗
- filter
- group by
- sum / mean / count
- TopK
- summary / insight / chart

---

## 运行

python3 csv_tool.py --input ../data/data.csv --group-by category --metric sum --field amount --filter city=Melbourne --k 3 --chart

---

## 目标

理解数据分析底层逻辑
