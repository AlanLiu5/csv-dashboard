# SQL 数据分析

使用 SQLite + SQL 实现数据分析。

---

## 功能

- CSV → SQLite
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- SUM / COUNT / AVG

---

## 运行

python3 sql_tool.py --input ../data/data.csv --group-by category --metric sum --field amount --filter city=Melbourne --k 2

---

## 目标

理解 SQL 与 pandas 的对应关系
